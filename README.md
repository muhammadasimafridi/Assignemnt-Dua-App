# 📿 Dua App

A simple and interactive **Dua Navigation Web App** built using HTML, CSS, and JavaScript. This app allows users to browse different duas using **Next** and **Previous** buttons, with an option to reset back to the starting dua.

---

## 🚀 Features

* ▶️ **Next Dua** – Navigate forward through duas
* ◀️ **Previous Dua** – Navigate backward through duas
* 🔄 **Start/Reset Option** – Return to the first dua
* 🖼️ **Dynamic Image Display** – Shows different dua images based on selection
* 🎯 **Range Control** – Prevents navigation beyond available duas

---

## 🧠 Concepts Used

* DOM Manipulation
* Event Handling (Click Events)
* Conditional Logic (if/else)
* State Management (tracking current index)
* Dynamic Content Updates

---

## 📂 Project Structure

* `HTML` → Layout and buttons
* `CSS` → Styling and alignment
* `JavaScript` → Navigation logic and image updates
* `Assets (Pics)` → Dua images

---

## 🎯 How It Works

1. App starts with the first dua displayed
2. User clicks **Next** → moves to the next dua
3. User clicks **Previous** → goes back to the previous dua
4. User clicks **Start** → resets to the first dua
5. Image updates dynamically based on current position

---

## 📌 Purpose

This project helps in understanding:
